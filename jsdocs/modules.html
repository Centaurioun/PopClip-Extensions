Action

Action: ActionDefinition | ActionFunction

An action can be either the ActionFunction on its own, or an ActionDefinition object.

If you supply the function on its own, the title and icon will be the extension name and extension icon.

LocalizableString

LocalizableString: string | object

A type to represent a localizable string.

Notes

The value may be either a string or an object. If you supply a string, that string is used. If you supply an object mapping language codes to strings, PopClip will display the string for the user's preferred language if possible, with fallback to the en string.

Example

 
option.label = "Color" // just use this string
option.label = { en: "Color", "en-GB": "Colour", fr: "Couleur", "zh-Hans": "颜色" }

define

  • define(objectOrFactory: object | (() => object)): void

  • Export an object for use by another file.

    Notes

    The define function exports an arbitrary object, which other files can import using require.

    It should be called only once in any file; if it is called more than once, only the final call will have any effect.

    It is used in the extension's Config.js file to define the extension itself.

    Example

     
    // greetings.js
    function hello(name) {    
      util.log("Hello, ${name}!");    
    })
    function goodbye(name) {    
      util.log("Bye, ${name}!");    
    })
    // export these two functions. hey, we made a module!
    define({hello, goodbye});

    Parameters

    • objectOrFactory: object | (() => object)

      Either the object to be defined, or a factory function which returns the object to be defined.

    Returns void

require

  • require(file: string): object

  • Import an object from another file.

    Example

     
    // load the functions from the greeting.js file
    const greetings = require('greetings.js');
    print(greetings.hello("PopClip")) // Hello, Popclip!

    // alternative using destructuring assignment
    const {hello, goodbye} = require('greetings.js');
    print(goodbye("PopClip")) // Bye, Popclip!

    Parameters

    • file: string

      Path to the file, relative to the root folder of the extension.

    Returns object

    The imported object.

Variables

pasteboard

pasteboard: PasteboardInterface

The global pasteboard object provides access to the contents of the macOS general pasteboard (i.e. the system clipboard). It implements PasteboardInterface.

popclip

popclip: PopClipInterface

The global popclip object encapsulates the user's current interaction with PopClip, and provides methods for performing various actions. It implements PopClipInterface.

util

util: UtilInterface

The global util object acts as a container for various utility functions and constants. It implements UtilInterface.
